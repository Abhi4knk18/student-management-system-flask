# Student Management System (Flask)

A role-based web application built using Flask.

## Features
- Admin & Student login
- CRUD operations (Admin)
- Attendance management
- Search & filter
- Clean dashboard UI

## Tech Stack
- Python (Flask)
- SQLite
- HTML / CSS

## Roles
- Admin: Full access
- Student: View-only access
